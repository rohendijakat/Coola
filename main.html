About Coola

Coola is the web interface for FridgAI household climatic control. It works with the FridgAI on-chain registry and the IceCool Python app.

  • Zones: define setpoints (0.1°C) and cooling preference per zone.
  • Readings: log temperature readings per zone; used for trends and suggested mode.
  • Schedules: override setpoints by block range (for future chain sync).
  • Hysteresis: dead bands to reduce compressor/heater cycling.

Data summary

Keyboard & tips

Use the tabs to switch between Dashboard, Zones, Readings, Schedules, Hysteresis, and Settings. All data is stored in your browser (localStorage). Export JSON or CSV to backup. Import CSV with columns: Zone ID, Setpoint (°C), Cooling (Yes/No), Label.

FridgAI contract uses setpoint in decicelsius (0.1°C). IceCool CLI can sync zones and readings; Coola is the web companion for local or demo use.

RPC & contract (optional)

For future integration with FridgAI contract: set RPC URL and contract address in Settings. IceCool app can use the same config to submit registerZone / recordSetpointReading transactions.

Decicelsius reference

FridgAI and IceCool use setpoints in decicelsius (tenths of °C). Examples: 220 = 22.0°C, 185 = 18.5°C, 250 = 25.0°C. Valid range 0–500 (0°C–50°C). Hysteresis bands and schedule windows also use the same scale where applicable.

Thermostat modes

Suggested mode (COOL / HEAT / HOLD) is derived from the last reading vs setpoint and hysteresis band. COOL: reading above high threshold. HEAT: reading below low threshold. HOLD: within band. Zone health (ok, hot, cold, stale, no_data) helps prioritize which zones need attention.

Version & storage

Coola v1. Data key: coola_fridgai_data. Settings key: coola_fridgai_settings. Clear data via Settings → Clear all data. Export JSON for full backup (zones, readings, schedules, hysteresis).

No server required: open this file in a browser or serve statically. Compatible with FridgAI contract (Solidity) and IceCool (Python) for household climatic control and AI temperature setpoint suggestions.
